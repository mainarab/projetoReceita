# Projeto Receita
Esse projeto é uma atividade do projeto transforme-se, a qual tem o objetivo de criar uma página de receitas para treinar o HTML.

Contém uma página principal (index) que redireciona para as demais páginas com suas respectivas receitas.
São 5 receitas, cada uma dentro de uma pasta, para treinar o uso do caminho dos links.
Cada receita contém imagem, ingredientes, modo de preparo, opção de voltar para a página inicial e o link da receita dentro do site original.
